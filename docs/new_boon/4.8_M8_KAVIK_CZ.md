# Milestone 8: kavik.cz

**Status**: Not Started

## Overview

Rewrite kavik.cz website entirely in Boon with physically-based 3D UI rendering via RayBox. Include an accessibility layer for screen readers and an HTML/content layer for SEO bots and AI. Optionally test Servo for prerendering (desktop app is M11).

---

## Success Criteria

- [ ] Website renders with RayBox 3D UI
- [ ] Accessibility layer works with screen readers
- [ ] SEO bots and AI receive semantic HTML/content
- [ ] Performance is production-ready
- [ ] Optional: Servo prerendering tested

---

## Dependencies

- **Requires**: M5 (Multi-File Modules), RayBox finished
- **Enables**: Real-world proof of Boon for production websites

---

## Key Features

| Feature | Description |
|---------|-------------|
| RayBox rendering | 3D physically-based UI |
| Accessibility | Screen reader support, ARIA |
| SEO layer | HTML content for bots |
| Responsive | Works on mobile and desktop |
| Servo prerendering | Optional: test static HTML generation |

---

## Rendering Layers

```
┌───────────────────────────────────────┐
│           User's Browser              │
├───────────────────────────────────────┤
│    RayBox (3D Visual Rendering)       │  ← What users see
├───────────────────────────────────────┤
│   Accessibility Layer (ARIA, Focus)   │  ← Screen readers
├───────────────────────────────────────┤
│   Content Layer (Semantic HTML)       │  ← SEO bots, AI
└───────────────────────────────────────┘
```
